=== SMI2 ===
Tags: smi2, crosspost, related posts
Requires at least: 2.5
Tested up to: 2.7
Stable tag: 0.3.2

Plugin integrates SMI2.RU services

== Description ==
http://smi2.ru/instruments/

== Installation ==
1. Upload `smi2` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Fill your auth data for SMI2 at 'Settings -> SMI2' admin page
4. Fill "The excerpt" field and check "Post on SMI2" checkbox before posting
